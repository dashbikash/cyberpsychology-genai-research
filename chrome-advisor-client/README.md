# Chrome Advisor Client

This Chrome extension is built with WXT (Web Extension Toolkit) and React.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running.

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18 or higher)
*   [npm](https://www.npmjs.com/)

### Installation

1.  Clone the repository:

    ```bash
    git clone <repository-url>
    cd chrome-advisor-client
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

### Development

Run the development server. This will automatically reload the extension in your browser when you make changes.

```bash
npm run dev
```

### Build

Build the extension for production:

```bash
npm run build
```

### Zip

Create a zip file for distribution:

```bash
npm run zip
```

## 📋 Browser Configuration

The extension is configured to run in development mode and will automatically reload when changes are detected.

*   **Run:** `npm run dev`
*   **Test in Firefox:** `npm run dev:firefox`

## 🛠️ Troubleshooting

If you encounter issues with the development runner or browser conflicts, refer to the WXT documentation for advanced configuration options.
